# Centos 6.3 Vagrant

## Requirements

- Git
- Ruby & RubyGems (or RVM)
- Vagrant
- VirtualBox 4.2.6

## Preparation

### Install Vagrant

    $ gem install vagrant

### Add CentOS 6.3 Base Box

This can be done via the project <https://github.com/spilth/veewee-centos63>

## Installation

    $ git clone https://github.com/spilth/vagrant-centos63
    $ cd vagrant-centos63
    $ vagrant up
    $ vagrant ssh
